ÜB·02

Charakter-Rig bauen

Eine Figur aus Einzelteilen zusammensetzen und drei Zustände geben: Idle, Walk, Jump. Diese Figur ist die Basis für praktisch den gesamten restlichen Kurs. Original: Charakter.fla — reines Rigging, kein einziger Zeile Code.

Verschachtelte Symbole → Objekt mit Einzelteilen

Im Original ist die Figur eine Hierarchie verschachtelter MovieClips: Figur enthält Body, Neck (→ Kopf), Shoulder_Left/Right (Oberarme), die wiederum Forearm_Left/Right (Unterarme) und Hand enthalten, plus zwei Leg-Symbole. Jedes Teil hat seinen eigenen Drehpunkt (registration point) — dreht man die Schulter, dreht sich der ganze Arm mit, ohne dass man Ober- und Unterarm getrennt bewegen muss.

Die Web-Entsprechung: ein CharacterRig-Objekt mit einem pose-Datensatz, der für jedes Körperteil einen eigenen Winkel trägt. Beim Zeichnen wird das genau nachgebaut — Canvas' translate() + rotate() vor jedem Segment entspricht exakt dem verschachtelten Drehpunkt-Prinzip:

Animate (verschachtelte Symbole) 
Figur
 └─ Shoulder_Left (eigener Drehpunkt)
     └─ Forearm_Left (eigener Drehpunkt)
         └─ Hand
JavaScript (verschachtelte Transforms) 
ctx.translate(shoulderX, shoulderY);
ctx.rotate(shoulderAngle);
// Oberarm zeichnen
ctx.translate(0, armLength);
ctx.rotate(forearmAngle);
// Unterarm + Hand zeichnen

Von Hand gebaute Keyframes → Sinusfunktionen

Im Original wurden die drei Bewegungsabläufe (Idle/Walk/Jump) mit vielen einzelnen Keyframes von Hand animiert. Für die Portierung reicht oft eine einzige Sinusfunktion pro Körperteil — der Walk-Zyklus zum Beispiel schwingt Arme und Beine gegenläufig, exakt das, was Math.sin(t * geschwindigkeit) von Natur aus liefert.

Der Sprung: eine einmalige Kurve statt einer Schleife

Idle und Walk laufen endlos in einer Schleife — der Sprung dagegen ist eine einmalige Bewegung, die danach zu Idle zurückkehrt. Das entspricht exakt der später (Übung 5) eingeführten Unterscheidung zwischen state = "playing" (Sprung läuft) und state = "stopped" — hier schon als jumpDoneCallback vorwegnehmend eingebaut.

Zwischen Idle/Walk/Jump wechseln und beobachten, wie sich Arme und Beine bewegen. Diese Figur wird ab Übung 3 mit echter Steuerung (Tastatur, Kollision, Level) ausgestattet.
