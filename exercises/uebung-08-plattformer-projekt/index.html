Plattformer-Projekt

Das Kursprojekt: alles aus den vorigen Übungen kommt hier zusammen — plus fallende Münzen und eine Punkteanzeige. Original: Main.as, CharacterController.as, CharacterAnimationController.as, CoinController.as, Collider.as/LevelBox.as/LevelStage.as.

Fünf Klassen, eine klare Verantwortung pro Klasse

Das Original teilt das Projekt sauber auf, genau wie es Übung 7 vorbereitet hat:

  • Main — Spielleitung: erzeugt Charaktere und Münzen, hält Punktestand und Tastatur-Zustand
  • CharacterController — Position, Bewegung, Schwerkraft, Level-/Münz-Kollision
  • CharacterAnimationController — reine Zustandsmaschine (Idle/Walk/Jump), kennt nichts von Physik
  • CoinController — fällt mit Schwerkraft, landet auf Plattformen, kann eingesammelt werden
  • Collider/LevelBox/LevelStage — melden Plattformen beim Spiel an

Diese Aufteilung wurde 1:1 in JS-Klassen übernommen — jede Klasse hat exakt dieselbe Verantwortung wie ihr AS3-Original.

Was neu dazukommt: Münzen mit eigener Physik

CoinController ist im Kern ein kleiner Cousin von CharacterController: dieselbe Schwerkraft, dieselbe Landungs-Erkennung gegen Level-Plattformen — nur ohne Steuerung. Einsammeln funktioniert über einen einfachen Abstandstest im Charakter-Update, jeder Charakter kann Münzen einsammeln, nicht nur der gerade ausgewählte.

JavaScript 
checkCoins(coins) {
  for (const c of coins) {
    if (c.collected) continue;
    if (Math.hypot(
        this.x - c.x,
        this.y - 30 - c.y) < 26) {
      c.collected = true;
      this.game.addScore(1);
    }
  }
}

Alles über denselben Event-Bus verbunden

Charaktere, Münzen und Plattformen kennen sich alle nicht direkt — sie melden sich beim gemeinsamen Game.bus an (registerCharacter, registerCoin, registerLevelElement), genau wie in Übung 5/6. Das zahlt sich hier aus: Neue Objekttypen (Münzen) hinzuzufügen brauchte keine einzige Änderung an bestehendem Code — nur eine neue Klasse und ein neuer Event-Name.

Einen der drei Charaktere anklicken, mit ←/→/Space steuern, alle 10 Münzen einsammeln, um zu gewinnen. Münzen fallen von oben und landen selbstständig auf den Plattformen.
